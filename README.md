# Healing Mart Mini Game Dist

GitHub Pages 배포용 미니게임 JavaScript 저장소입니다.

## 폴더 구조

```text
hm-mini-game-dist/
├─ hm-hit-or-miss.4e01c173.min.js
├─ blogger/
│  └─ hm-hit-or-miss-call.html
└─ README.md
```

## Blogger 호출 코드

```html
<script src="https://healingmart.github.io/hm-mini-game-dist/hm-hit-or-miss.4e01c173.min.js" defer></script>
```

JavaScript 파일은 저장소 최상단에 배치합니다. `dist` 폴더는 사용하지 않습니다.
